### <span style="color:#F2C759">✦ Play</span>

1. <span style="color:#F2C759"><b>Find</b></span> relics in towers, camps, chests, mimics, and archaeology.
2. <span style="color:#F2C759"><b>Equip</b></span> them with the Relic Crate / Reliquary.
3. <span style="color:#C44AA8"><b>Boost</b></span> Might, Ward, Gale, Fortune, Vitality, or Alchemy from your loadout.
4. <span style="color:#C44AA8"><b>Attune</b></span> skills at the Attunement Forge.

### <span style="color:#F2C759">✦ Mimics</span>

Six biome-tinted chest mimics (Forest, Jungle, Snow, Swamp, Desert, Badlands). Ambush deep/camp chests — defeat them for <b>Relic Dust</b> and nearby relic loot.

![Swamp Mimic](https://raw.githubusercontent.com/DeepForge-Studios/RPG-Relics/main/site/media/mimics/swamp.png)

### <span style="color:#F2C759">✦ Start</span>

Import the `.mcaddon`, enable Behavior + Resource packs, leave Experiments / Beta APIs off for achievements. First join gives a Relic Tome, Relic Crate, starter relics, and materials.
