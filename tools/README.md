# Tools

Optional utilities for pack authors. Players only need `build_mcaddon.ps1` at the repo root.

| Script | Purpose |
|--------|---------|
| `gen_curio_form_ui.py` | Regenerate Reliquary / form UI textures |
| `gen_held_attachables.py` | Held-item attachable helpers |
| `gen_mimic_models.py` / `gen_mimic_variants.py` | Mimic model / skin variants |
| `gen_relic_tower.py` / `import_relic_tower.py` | Tower structure helpers |
| `strip_item_groups.mjs` | Strip item `menu_category.group` for catalog |

Requires Python 3 and/or Node as noted in each script.
